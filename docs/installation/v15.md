## Setup database user
```
sudo mariadb
CREATE USER 'frappe'@'127.0.0.1' IDENTIFIED BY '<password>';
GRANT ALL PRIVILEGES ON *.* TO 'frappe'@'127.0.0.1' WITH GRANT OPTION;
FLUSH PRIVILEGES;

CREATE USER 'root'@'127.0.0.1' IDENTIFIED BY '<password>';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'127.0.0.1' WITH GRANT OPTION;
FLUSH PRIVILEGES;
```

### Alter Host
```
ALTER USER 'frappe'@'127.0.0.1' IDENTIFIED BY '<password>';
FLUSH PRIVILEGES;

ALTER USER 'root'@'127.0.0.1' IDENTIFIED BY '<password>';
FLUSH PRIVILEGES;
```

### View Host
```
SELECT user, host FROM mysql.user WHERE user='frappe';
```

# Migrate Private and Public Files
## Archive Files
```
tar -cvf private_file.tar sites/<site-name>/private
tar -cvf public_file.tar sites/<site-name>/public
```

## Copy files to live server

## Restore Files
```
bench restore --with-public-files <public_tar_file_path>
bench restore --with-private-files <private_tar_file_path>
or 
tar -xvf files.tar -C /destination
or 
cp -r . /home/ubuntu/frappe-bench/sites/<site_name>/private/files/
or (best)
rsync -av private_files/ /home/ubuntu/frappe-bench/sites/<site_name>/private/files/
```

## Copy Required Config Of site_config.json
```
bench set-config <key> <value>
```
